# Student Nutrition & Performance Analysis

## Overview
This project explores relationships between student academic performance and nutrition using Python and data analysis techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- PySpark
- Jupyter Notebook

## Features
- Data preprocessing
- Dataset integration
- Statistical analysis
- Data visualization
- Insight generation

## Datasets
- Student Performance Dataset
- Diet Recommendation Dataset

## Results
The analysis generates insights into how nutrition-related factors may influence student performance and learning outcomes.
